## Getting Start

### Requirements

컴퓨터에 **Python 3** 버전이 설치되어야 합니다. 

### Usage

해당 레포지토리를 clone으로 내려받고 프로젝트 경로에서 `main.py`를 실행하세요. 
```
$ git clone https://github.com/nuxlear/evil_hunter_calculator.git
$ cd evil_hunter_calculator
$ python main.py
```
